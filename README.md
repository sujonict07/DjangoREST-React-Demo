# Using Ubuntu

$ curl -sL https://deb.nodesource.com/setup_11.x | sudo -E bash -
$ sudo apt-get install -y nodejs

$ sudo npm i -g create-react-app
$ create-react-app <appname>
  its zero configurtaion for you, becouse, development server, webpack, 
  and babel are automatically installed on create-react-app commands.Although you want 
  to custom configure, then use
$ npm run reject
Now, 
$ cd <appName>
$ npm start 
  it will lunch a development server with 3000 port.
  that means http://localhost:3000
